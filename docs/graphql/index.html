EmbedServer GraphQL API

Upload documents (PDF, DOCX, TXT, MD), chunk and embed them via Ollama, store in Qdrant, and search by semantic similarity.

Queries

search

Response

Returns a SearchResponse!

Arguments
Name Description
query - String!
limit - Int Default = 5

Example

Query
query search(
  $query: String!,
  $limit: Int
) {
  search(
    query: $query,
    limit: $limit
  ) {
    query
    results {
      ...SearchResultFragment
    }
  }
}
Variables
{"query": "abc123", "limit": 5}
Response
{
  "data": {
    "search": {
      "query": "xyz789",
      "results": [SearchResult]
    }
  }
}
